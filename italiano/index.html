Istruzioni per l'installazione

  1. Scarica l'archivio zip cliccando il tasto Scarica Ora
  2. Estrai l'archivio .zip
  3. Copia tutti i file presenti nella cartella "TinyUpper" e copiali all'interno del tuo server
  4. Modifica il file upload.html
  5. Modifica il file Upload.php
  6. Crea all'interno del tuo server la cartella dove vuoi che i file siano caricati
  7. É consigliabile copiare i file di TinyUpper nella stessa directory in cui é stata creata la cartella
    8.
  8. Inserisci all'interno della cartella di Upload il contenuto della cartella "elenconaut"
  9. Elenconaut serve per visualizzare i file che sono contenuti all'interno della tua cartella di Upload
    10.

Lista Caratteristiche

  • Interfaccia semplice e pulita
  • Grafica leggera per funzionare pure su dispositivi non recenti
  • Supporto per Android, Windows e Mac Os
  • Disponibile in Inglese e Italiano (in futuro Tedesco e Cinese.)
  • Script in php veloce e sicuro
  • Funzionante con la maggior parte dei server
  • Personalizzabile al 100%

Requisiti

  • PHP 5.x e/o superiori
  • My SQL come server management
  • In Alternativa Server Apache
  • Dominio & Hosting
  • Spazio su server superiore a 1GB
  • Client FTP Filezilla
